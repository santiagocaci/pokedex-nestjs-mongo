# Pokedex

## Installation

```bash
$ npm install

$ npm install -g @nestjs/cli

# docker-compose up -d
$ npm run docker:up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Seed

```bash
# 151 pokemons
http://localhost:3000/api/v2/seed
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
